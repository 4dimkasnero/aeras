Ατμόσφαιρα

Τμήμα ΣΤ1

Όνομα ομάδας: Τα Τσακάλια

Μαθητές:


Συντονιστής ομάδας: Καθηγητής Πληροφορικής Μήλιος Αθανάσιος.

Στόχος της εργασίας μας είναι να πάρουμε κάποιες μετήσεις απο την ατμόσφαιρα όπως
1)Θερμοκρασία
2)Υγρασία
3)Μονοξειδίο Άνθρακα
4)Ποιότητας Αέρα (VOC&eCO2)
5)Υπεριώδη Ακτινοβολία

Τα υλικά που θα χρειαστούμε είναι:
1)arduino
2)Αισθητήρα Θερμοκρασίας/Υγρασίας/Βαρομετρικής Πίεσης
3)Αισθητήρα Μονοξειδίου Άνθρακα
4)Αισθητήρα Ποιότητας Αέρα (VOC&eCO2)
5)Αισθητήρα Υπεριώδους Ακτινοβολίας
